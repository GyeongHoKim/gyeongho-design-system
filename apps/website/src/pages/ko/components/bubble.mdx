`Bubble`은 현재 **React 전용** 컴포넌트예요(`@ghds/react`). 아직 Web Components나 React Native 대응물은 없어요.

## 개요

`Bubble`은 단일 채팅 말풍선이에요 — 메시지 텍스트를 담는 스케치 둥근 박스. 두 가지 변형이 있어요: `received`(은은한 수신 말풍선)와 `sent`(주요 색상의 채워진 발신 말풍선). 자신만 그려요; 대화 내 정렬은 호출자의 몫이에요(`Message` 컴포넌트 참고).

## 구조

메시지 텍스트를 감싸는 손그림 둥근 박스(`@ghds/sketch-core`의 외곽선 + 채우기)예요. 색상, 패딩, 반경, 스케치 파라미터는 `@ghds/tokens`(`comp.bubble.*`)에서 와요. 박스는 `inline-block`이고 `max-width: 100%`이므로, 긴 텍스트는 넘치지 않고 줄바꿈돼요.

## 변형 및 상태

두 가지 변형, 인터랙티브 상태 없음 — 말풍선은 정적 콘텐츠예요.

## 접근성

- 말풍선은 일반 컨테이너예요; 의미는 텍스트 콘텐츠예요. 색상이 sent와 received를 구분하지만, 대화 구조(누가 무엇을 말했는지)는 색상만이 아니라 `Message`의 작성자와 레이아웃에서 나와야 해요.
- 스케치 박스는 장식용이며 알려지는 내용을 절대 바꾸지 않아요.
- 공통 관례는 <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.

## 콘텐츠

말풍선 콘텐츠는 메시지 자체로 유지하세요. 작성자와 타임스탬프는 말풍선 안이 아니라 감싸는 `Message` 헤더에 속해요.
