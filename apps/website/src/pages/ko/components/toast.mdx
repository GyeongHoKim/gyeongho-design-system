## 개요

`Toast`는 페이지 위(우하단)에 플로팅되고 자동으로 사라지는 짧은 일시적 알림이에요. 확인과 낮은 우선순위
피드백에 사용하세요; 레이아웃 내에 지속되는 상태에는 <a href={withBase('/components/alert/')}>Alert</a>를
사용하세요.

## 구조

우하단에 고정된 스케치 스타일의 떠 있는 박스(`@ghds/sketch-core`)로, 심각도 아이콘, 선택적 제목, 메시지,
닫기 버튼이 있어요. `duration`(ms; `0`이면 지속) 후 자동으로 사라져요. `open`/`onClose`로 제어하고,
여러 개를 렌더링해서 쌓을 수 있어요.

## 접근성

- **모든 플랫폼**: `danger` 토스트는 `role="alert"`(단정적)을 사용해요; 나머지는 `role="status"`
  (정중) / `aria-live` 영역을 사용해서, 포커스를 빼앗지 않고 알려져요. 라벨이 있는 "닫기" 버튼이 항상 있어요.
- **React Native**: 투명한 `Modal`에 `pointerEvents="box-none"`으로 렌더링돼서, 토스트 외부의 탭이
  통과해요.
- <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.

## 콘텐츠

짧은 메시지와 선택적 제목. 자동 사라짐 기본값은 5000ms예요 (동작 토큰이 아닌 행동 타이밍이에요);
닫을 때까지 유지하려면 `duration={0}`을 전달하세요.
