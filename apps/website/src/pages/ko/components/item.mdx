`Item`과 그 하위 컴포넌트는 **React 전용**이에요(`@ghds/react`). 아직 Web Components나 React Native 대응물은 없어요.

## 개요

`Item`은 유연한 목록 행 프리미티브예요: 선택적 선행 미디어, 늘어나는 콘텐츠 열, 후행 액션의 가로 띠예요. `ItemMedia`, `ItemContent`, `ItemTitle`, `ItemDescription`, `ItemActions`로 조합하세요. 설정 행, 멤버 목록, 선택 가능한 옵션, 유사한 목록형 UI에 사용하세요.

## 구조

세 개의 선택적 슬롯을 좌에서 우로 배치하는 flex 행(`<div>`)이에요:

- **`ItemMedia`** — 아이콘, 아바타, 썸네일을 위한 고정된 선행 슬롯.
- **`ItemContent`** — 늘어나는 가운데 열로, 보통 `ItemTitle` 위에 `ItemDescription`이 있어요.
- **`ItemActions`** — 버튼, 배지, 셰브론을 위한 고정된 후행 슬롯.

`outline` 변형은 `@ghds/sketch-core`를 통해 손그림 테두리를 그려요; 모든 색상, 간격, 스케치 파라미터는 `@ghds/tokens`(`comp.item.*`)에서 와요.

## 변형 및 상태

세 가지 표면 변형 — `default`(투명), `muted`(은은한 채우기), `outline`(손그림 테두리) — 에 더해 선택된 배경을 채우는 `selected` 불린이 있어요.

## 접근성

- `Item`은 암시적 role이 없는 일반 `<div>`를 렌더해요 — 조합에 필요한 대로 컨테이너에 `role="listitem"`, `role="option"` 등을 설정하고, 일치하는 컨테이너 role(`list`, `listbox`)로 감싸세요.
- 행이 무언가를 활성화하게 만들 때 `tabIndex`, `button` 같은 role, 키보드 처리를 추가하세요 — 어느 것도 기본 제공되지 않아요.
- 선택 상태가 그려지기만 하지 않고 알려지도록 `selected` prop을 `aria-selected`와 함께 사용하세요.
- 공통 관례는 <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.

## 콘텐츠

`ItemTitle`은 주요 줄이에요 — 문장이 아닌 짧은 라벨. `ItemDescription`은 보조 정보를 위한 은은한 두 번째 줄이에요. 행이 스캔하기 좋도록 미디어는 아이콘/아바타 하나로, 액션은 한두 개의 컨트롤로 유지하세요.

```tsx
import { Item, ItemMedia, ItemContent, ItemTitle, ItemDescription, ItemActions } from '@ghds/react/item';
import { Badge } from '@ghds/react/badge';

<Item variant="outline">
  <ItemMedia>{/* avatar */}</ItemMedia>
  <ItemContent>
    <ItemTitle>GyeongHo Kim</ItemTitle>
    <ItemDescription>Maintainer · 3 open PRs</ItemDescription>
  </ItemContent>
  <ItemActions>
    <Badge variant="success">Online</Badge>
  </ItemActions>
</Item>;
```
