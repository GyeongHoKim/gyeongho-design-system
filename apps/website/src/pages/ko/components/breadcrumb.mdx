## 개요

`Breadcrumb`은 사이트 계층 구조에서 현재 페이지의 위치를 보여주고, 사람들이 경로를 따라 상위로 이동할 수 있게 해줘요. 보조 내비게이션 보조 도구예요 — 주 내비게이션을 대체하지 않아요.

## 구조

정렬된 목록을 감싸는 `<nav>` 랜드마크예요: 각 상위 항목은 링크이고, 손그림 셰브론으로 구분되며, 마지막 항목은 현재 페이지(링크 아님)예요. API는 모든 플랫폼에서 **데이터 기반**이에요 — `{ label, href? }` 형태의 `items` 배열을 전달하세요.

## 접근성

- **React / Web Components**: `<nav aria-label="Breadcrumb">` 랜드마크가 `<ol>`을 감싸고, 현재 페이지는 `aria-current="page"`를 가져요. Web Components는 링크 활성화 시 `select` 이벤트를 발생시켜요.
- **React Native**: 행은 라벨과 함께 `role="navigation"`을 노출하고, 항목은 `accessibilityRole="link"`예요. React Native에는 `href`가 없어요 — `onSelect` 콜백에서 내비게이션을 처리하세요.
- <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.

## 콘텐츠

목적지 페이지 제목을 반영하는 짧고 인식하기 쉬운 라벨이에요. 마지막 항목은 현재 페이지의 이름이에요.
