## 개요

`Pagination`은 긴 결과 목록을 번호가 매겨진 페이지로 나누고, Previous/Next 컨트롤과 큰 범위를 접는 말줄임표를 제공해요. 제어 방식이에요 — 현재 `page`를 소유하고 `onPageChange`에서 업데이트해요.

## 구조

Previous, 번호가 매겨진 페이지 버튼들, Next를 감싸는 `<nav>` 랜드마크예요. 각 버튼은 손그림 박스(`@ghds/sketch-core`)이며, 현재 페이지는 채워지고 `aria-current="page"`로 표시돼요. 큰 간격은 단일 말줄임표로 접혀서 첫 번째, 마지막, 현재 페이지 주변 페이지(`siblingCount`)를 유지해요.

## 접근성

- **React / Web Components**: `<nav aria-label="Pagination">` 랜드마크; 각 컨트롤은 설명적인 `aria-label`("Page 3", "Previous page")을 가진 실제 `<button>`; 현재 페이지는 `aria-current="page"`를 가져요. Web Components는 `page-change` 이벤트를 발생시켜요.
- **React Native**: 행은 `role="navigation"`을 노출; 각 컨트롤은 `accessibilityRole="button"`에 라벨과 `accessibilityState`(현재 페이지는 `selected`, 끝은 `disabled`)를 가져요.
- 웹 플랫폼에서 모든 컨트롤은 키보드로 조작 가능해요.
