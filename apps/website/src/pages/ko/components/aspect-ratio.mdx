`AspectRatio`는 현재 **React 전용** 프리미티브예요(`@ghds/react`). 디자인 값을 소유하지 않고 아무것도 그리지 않으므로, 아직 Web Components나 React Native 대응물은 없어요.

## 개요

`AspectRatio`는 네이티브 CSS `aspect-ratio` 속성을 사용해 고정된 가로세로 비율의 박스를 확보해요 — 대표적인 용도는 반응형 미디어(이미지, 비디오, 지도 임베드)를 담아 에셋이 로드되는 동안 레이아웃이 흔들리지 않게 하는 거예요. 사용 가능한 너비를 채우고, `ratio`로부터 높이를 도출해요.

## 구조

`width: 100%`, 주어진 `aspect-ratio`, `overflow: hidden`으로 설정된 단일 `<div>`예요. 단일 자식을 `width: 100%; height: 100%`로 크기 지정하면(예: `object-fit: cover`인 `<img>`) 박스를 채워요. 스케치 레이어는 없어요 — 이 프리미티브는 순전히 구조적이에요.

## 변형 및 상태

AspectRatio는 **정적 레이아웃 박스**예요 — 인터랙티브 상태가 없어요. 유일한 변화는 `ratio` 값 자체예요.

## 접근성

- role이나 label이 없는 일반 `<div>`를 렌더해요 — 레이아웃 래퍼이므로, 접근 가능한 콘텐츠는 안에 넣는 자식(예: `<img>`는 자체 `alt`가 필요해요)이에요.
- 미디어가 로드되기 전에 공간을 확보하므로 레이아웃 이동을 방지해요 — 느린 연결의 사용자에게 실질적인 접근성 및 사용성 이점이에요.
- 공통 관례는 <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.
