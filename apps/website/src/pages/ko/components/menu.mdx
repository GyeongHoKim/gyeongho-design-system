## 개요

`Menu`(드롭다운)은 버튼으로 실행되는 **동작** 목록을 제공해요 — Edit, Duplicate, Delete. 목록에서 지속적인 값을 선택하려면 <a href={withBase('/components/select/')}>Select</a>를 대신 사용하세요.

## 구조

`aria-haspopup="menu"`를 가진 `<button>` 트리거가 `role="menuitem"`들의 떠 있는 `role="menu"` 패널을 열어요. 트리거와 패널 모두 손그림 박스(`@ghds/sketch-core`)예요. 데이터 기반 — `{ value, label, disabled? }` 형태의 `items` 배열을 전달하세요.

## 접근성

- **React / Web Components**: `aria-haspopup="menu"` + `aria-expanded` 트리거; `role="menu"` 패널의 `role="menuitem"`. Enter/Space/ArrowDown이 열고 첫 항목에 포커스; Arrow/Home/End로 포커스 이동; Enter/Space로 활성화; Escape로 닫고 트리거로 포커스 복원; 외부 클릭으로 닫아요. Web Components는 `select` 이벤트를 발생시켜요.
- **React Native**: `Modal`에 고정된 패널; 탭으로 선택. `accessibilityRole="menu"`/`"menuitem"`. 화살표 키 roving은 웹 전용이에요.
- <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.
