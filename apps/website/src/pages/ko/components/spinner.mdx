## 개요

`Spinner`는 비결정형 로딩 표시자예요 — 지속 시간을 알 수 없는 작업이 진행 중일 때 회전하는 작은
손그림 스타일의 링이에요. 의미 있는 진행률을 보여줄 수 없을 때 사용하고, 보여줄 수 있을 때는
<a href={withBase('/components/progress/')}>Progress</a>를 사용하세요.

## 구조

기본 색상으로 렌더링되고 회전하는 스케치 스타일의 타원 링(`@ghds/sketch-core`)이에요. 윤곽선이 손그림
스타일이고 불규칙하기 때문에, 간격이나 그라데이션 없이도 회전이 명확하게 보여요. 세 가지 크기 —
`sm`, `md`(기본), `lg` — 가 `comp.spinner.size.*`에 매핑되고, 회전 지속 시간은
`comp.spinner.duration`이에요.

## 접근성

- **React**: `role="status"`와 `aria-label`(기본값 "Loading")을 사용해서 로딩 상태를 알려요.
  회전하는 SVG는 `aria-hidden`이에요.
- **Web Components**: `gh-spinner`는 `internals.role = 'status'`와 `internals.ariaLabel`을 설정해요.
- **React Native**: `accessibilityLabel`과 `accessibilityState={{ busy: true }}`를 설정해요 (React Native에는
  `status` 역할이 없어요).
- **동작 감소**: 세 플랫폼 모두 사용자가 동작 감소를 요청하면 회전을 숨겨요
  (웹에서 `prefers-reduced-motion`; React Native에서 OS "동작 줄이기" 설정).
- <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.
