## 개요

`Table`은 정렬 가능한 열과 선택적 행 선택 기능이 있는 구조화된 데이터 행을 표시해요. 정렬과 선택은
**제어되는** 방식이에요 — 테이블은 전달된 `rows`를 전달된 순서대로 렌더링하고, `onSortChange` /
`onSelectionChange`를 통해 의도를 전달해요.

## 구조

스케치 스타일의 윤곽선(`@ghds/sketch-core`)이 시맨틱 `<table>`(React Native에서는 플렉스 행)을
감싸요. 정렬 가능한 헤더는 `aria-sort`를 가진 버튼이고, 선택적 앞 체크박스 열(모두 선택 헤더 포함)이
선택을 담당해요. 이건 M12 데이터 상태 패턴의 데이터 기반이에요.

## 접근성

- **React / Web Components**: 시맨틱 `<table>`에 `<caption>`, `scope="col"` 헤더, 정렬 가능한 헤더에
  `aria-sort`(`none`/`ascending`/`descending`)가 있어요. 선택은 접근성 라벨이 있는 GHDS Checkbox를
  사용해요 ("행 선택 …", "모든 행 선택"). Web Components는 `sort-change`와 `selection-change`를
  디스패치해요.
- **React Native**: 행은 `role="table"`/`row`/`columnheader`/`cell`을 가진 플렉스 `Box`예요; 정렬은
  헤더 버튼, 선택은 `role="checkbox"` 컨트롤이에요. `aria-sort`는 문서화된 웹 전용 차이예요.
- <a href={withBase('/accessibility/')}>접근성 가이드</a>를 참고하세요.
